# RLGym Cartpole

This is an implementation of the [RLGym](https://github.com/lucas-emery/rocket-league-gym) API for the famous environment [CartPole](https://gymnasium.farama.org/environments/classic_control/cart_pole).

## Installation

You can install this package with the following command

```python
pip install rlgym-extra-envs[cartpole] # Everything but the render
pip install rlgym-cartpole # Everything but the render
pip install rlgym-cartpole[render] # Everything
```
