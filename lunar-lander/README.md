# RLGym Lunar Lander

This is an implementation of the [RLGym](https://github.com/lucas-emery/rocket-league-gym) API for the famous environment [Lunar Lander](https://gymnasium.farama.org/environments/box2d/lunar_lander/).

## Installation

You can install this package with the following command

```python
pip install rlgym-extra-envs[lunar-lander] # Everything but the render
pip install rlgym-lunar-lander # Everything but the render
pip install rlgym-lunar-lander[render] # Everything
```
